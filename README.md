
# EC2 Apache RedHat Infrastructure (Terraform)

This project provisions a scalable AWS infrastructure using Terraform and deploys an Apache-based application on a RedHat EC2 instance.

---

## 🌐 Live Project Links

- 🔴 RedHat Infrastructure Overview:  
  https://www.fejzic37.com/links/redhat.html

---

## ⚙️ What this project does

- Provisions EC2 instances on AWS
- Configures security groups and networking
- Deploys Apache HTTP Server on RedHat
- Automates infrastructure using Terraform
- Includes reusable infrastructure components

---

## 🧱 Tech Stack

- AWS (EC2, VPC, Security Groups)
- Terraform (Infrastructure as Code)
- RedHat Linux
- Apache Web Server

### ☁️ Cloud & Infrastructure
- AWS EC2 (RedHat Linux)
- AWS VPC (custom networking)
- AWS Load Balancer (ALB)
- AWS Security Groups
- Bastion Host architecture
- CloudWatch Monitoring

### 💻 Application Layer
- Java (Servlets + JSP)
- Apache Tomcat / Apache HTTP Server
- Maven build system

### 🗄️ Database
- PostgreSQL
- JDBC connectivity
- Relational schema for message storage

### 🔐 Security
- TLS/SSL encryption (HTTPS)
- Bastion-host-based SSH access control
- IAM-based AWS access control concepts

### 📊 Observability
- AWS CloudWatch logs & metrics
- System-level monitoring
